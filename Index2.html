STUDIO FINAL PROJECT

\ \

Inspiration

\ \ \

Inspired the Virtual Reality game Beat Saber, which the player cutting the music note to win score. I want to explore and make a music-related Augmented Reality game. Different from Virtual Reality headset that created an immersive experience, Augmented Reality enable the player to remain in the real word and interact with virtual objects. For this project, I will use Microsoft Hololens as the display headset and Unity to program the game. \

\ \ \ \ \ \

Also inspired by the sport dodgeball that player moving left and right to avoid hit. I want to combine it with the music aspect to create a game that the player needed to avoid hit by music note.\

\ \

Research & Ideas

\ \

For making this project, I did a little bit research to see what is possible. After learning the basic connection of Microsoft Hololens and Unity, I had a better understanding of the devices. Secondly, I research for the music visualizer, which can automatically generated waves/ object movement by inputting a song. I will use the music visualizer as the producer of the notes, so that when the notes will be fit with the song when generated. Moreover, I saw some very interesting augmented reality board game that used a target object and display the character on top. In this way, there will be not only the virtual objects, but also physical ones that player can actually move around, which can add more fun experience to the game.
